PGRN 项目统一启动目录

以后只使用这个文件夹里的入口。可双击文件：

1. 打开完整版.cmd
   用途：打开桌面完整版。
   特点：如果源码比当前 `pgrn.exe` 新，会先自动重建再启动；这里默认只重建可运行 exe，不额外打 installer。
   额外规则：如果当前已经有 `pgrn` 进程占用 exe，会先关闭旧进程，再启动最新版本。

2. 开发模式.cmd
   用途：启动 Tauri 开发模式。

3. 构建完整版.cmd
   用途：手动重新构建最新版桌面程序。
   构建完成后，会把最新安装包自动同步到 `启动入口\发布\`。

4. 发布\
   用途：统一存放可发给别人安装的安装包。
   当前推荐优先使用这里的 `.msi` 或 `-setup.exe`。

规则：
- 根目录下旧的 bat/cmd 只是兼容转发壳，不是主入口。
- 以后所有启动相关修改，只应改这个文件夹。
- 如果 AI 后续修改了启动方式，也必须同步更新这个文件夹。
